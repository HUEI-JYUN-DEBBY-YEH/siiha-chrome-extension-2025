Privacy & Permissions

  • Local-first: All features run on-device. We do not read or upload page inputs.
  • Storage: Data is stored in chrome.storage.local. You can clear it anytime in Options.
  • Hybrid: Off by default. If enabled later, only anonymized daily summaries may be sent; never verbatim text.
  • Permissions: storage, alarms, idle, activeTab for UI placement.

This is a developer preview (v0.1). Crisis resources are offline JSON in later versions.
